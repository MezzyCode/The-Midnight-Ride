Simple Fallout 4 Downgrader

Reverts game code to version 1.10.163 for wider modding support.

Installation Instructions:
  1. Main Files - Simple Fallout 4 Downgrader
  2. From the downloaded archive, extract everything to the game's Root folder.
  3. Double-click the downgrader executable to launch it.
  4. Wait for the "Patching successful" message in the console window that just opened, then you can close it.

Creating a Separator in MO2.

Now that we have some mods installed, it's time to create our first separator:

  1. Right-click the empty space in the left pane of MO2 and select Create separator.
  2. Name the separator Utilities.

High FPS Physics Fix

All-in-one solution for smooth FPS-unlocked gameplay, mitigates stutter and reduces loading times.

Installation Instructions:
  • Main Files - High FPS Physics Fix 0.8.6 (Click the blue name above for a direct link to the correct version)
  • Once the mod has been installed, double-click on it in the left pane of MO2
  • In the INI-Files tab, select F4SE/Plugins/HighFPSPhysicsFix.ini
  • Change the following options:
    • Set AllowTearing to true (line 26)
      Enabling AllowTearing allows the mod to disable VSync during loading in order to make it faster, since the game loading time is largely limited by current framerate (up to a certain point). Enabling it will NOT lead to any tearing during gameplay. This setting is also a requirement for VRR.
    • Set DisableBlackLoadingScreens to true (line 37)
    • Set DisableAnimationOnLoadingScreens to true (line 44)
      Enabling DisableAnimationOnLoadingScreens will freeze loading screens and any overlay you use, this is intended and will lead to way faster loading times. You can omit it if you prefer looking at animated loading screens.
    • Set PostloadingMenuSpeed to 3.0 (line 51)
    • If you use VRR (GSync/FreeSync), set InGameFPS (line 58) to any value close to your lowest FPS and not exceeding your refresh rate - (refresh rate x 6.8%)

      You can check if you are using VRR by going to the Nvidia Control Panel if you're on Nvidia or Adrenalin's Display Settings if you're on AMD. If the GSync tab is missing from NVCP, your display doesn't support it or doesn't have the feature enabled in OSD.

      NVIDIA USERS ONLY: Make sure that GSync is set to Fullscreen and that Display Specific settings are enabled in the Control Panel, the Windowed option is problematic due to it being a hacky method. Both of the settings are wrongly named so they do not actually directly refer to the display method.

    • If you use fixed refresh rate (meaning no VRR), set InGameFPS (line 58) to your refresh rate - 0.005 or fractions of your refresh rate. For example on 60hz you can lock to 59.995 or 30. Just like on VRR, choose whether to lock to a fraction or RR-0.005 based on which is closest to your lowest FPS.
    • If you set InGameFPS to a value higher than 60, do the following as well:
      • Set DynamicUpdateBudget to true (line 264)
      • Set BudgetMaxFPS to the same value as InGameFPS (line 283)

IMPORTANT: Make sure that no other external program such as RTSS or your GPU's drivers are applying a limiter or forcing VSync. These features have to be handled by the mod in order to achieve the best results.
