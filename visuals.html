Visuals

In this step, we will install mods improving the overall look and feel of the game, without any drastic or performance demanding changes

Creating a Separator in MO2.

  1. Right-click the empty space in the left pane of MO2 and select Create separator.
  2. Name the separator Visuals.

Lightweight Lighting - A weather and interior lighting overhaul

Installation instructions:

  • Main Files - Lightweight Lighting
Lightweight lighting aims to be an all in one simple lighting overhaul while being as compatible as can be with no cell edits.

Gloomy Glass - All transparent materials revised

Installation instructions:

  • Main Files - Gloomy Glass
Balances the environmental effects on all base game and DLC transparent material files.

Targeted Textures - Vanilla textures upscaled fixed and sized

Installation instructions:

  • Main Files - Targeted Textures
Fixes numerous texture sizing, coloration and specular errors, with the help of AI upscaling and other sources from around the Nexus.

Fallout 4 Particle Patch - No More Glowing Objects

Installation instructions:

  • Main Files - FO4 Particle Patch v1-2 Main
Edits hundreds of mesh and material files to make fog, smoke, water, blood, dirt puffs, dust (and much more) glow in the dark less, and better utilize ambient & point lighting. Also improves distance rendering for Far Harbor fog effects.

WET - Water Enhancement Textures

Installation instructions:

  • Main Files - WET - Water Enhancement Textures
  • You will be prompted with a FOMOD installer:
    1. WET Assets With Rain
    2. Install
Enhances all water and most water-related effects in the game.

WAVE - Wave Animation Vanilla Enhanced

Installation instructions:

  • Main Files - WAVE
Brand new wave effects and textures with a more realistic motion.

Enhanced Vanilla Armor and Clothing - EVAC

Installation instructions:

  • Main Files - Enhanced Vanilla Armor and Clothing 2k
Complete overhaul of the appearance of armor and clothing.

Hair Specular Map Removal Thingy

Installation instructions:

  • Main Files - Hair Specular Map Removal Thingy
Fixes glowing hair when under certain lighting.

First-Person Swimming Animations Tweak

Installation instructions:

  • Main Files - New Swimming Anims - Standard
Adds arms to the first person swimming animations.

First-Person Running with Hands Animations

Installation instructions:

  • Main Files - Running Hands ESL 1.1
Adds arms to the first person running animations.

Sniper Scope Overlay And Reticles Replacement

Installation instructions:

  • Main Files - Depth of Field Effect ESP
  • Main Files - Sniper Scope Overlay And Reticles Replacement
Gets rid of the black borders around the reticle and puts a scope overlay in its place. It also changes the reticles.

Diamond City Billboards

Installation instructions:

  • Main Files - Diamond City Billboards
Places the billboards and Nuka-Cola ad you can see when inside the city, on the outside.

Diamond City Supplements

Installation instructions:

  • Main Files - Diamond City Supplements
Fleshes out Diamond City's empty stadium seats with new buildings and shacks.

Goodneighbor View

Installation instructions:

  • Main Files - Goodneighbor View
Adjusts Goodneighbor so that the inside and outside of the town better match each other.

Remove Ironsight Blur

Installation instructions:

  • Main Files - RemoveBlur
Removes the blurring effect in weapon sights.

Terrain Undersides - 2 Million Sunblock

Installation instructions:

  • Main Files - Terrain Undersides
Block sun rays shining through the ground.
